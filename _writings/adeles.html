1 Restricted products

Let I be a nonempty set and for i∈I suppose that Xi is a locally compact space and that Ki is a compact open set in Ki. A subset J of I is said to be almost all I if I∖J is finite. Define the restricted product

X=∏^i∈IKi⁢Xi={x∈∏i∈IXi:xi∈Ki for almost all i∈I}.

The restricted product topology is the topology τ on X generated by the collection ℬ of sets of the form ∏i∈EUi×∏i∉EKi where E⊂I is finite and for each i∈E, Ui is an open set in Xi.

Lemma 1.

ℬ is a base for τ.

Proof.

For B1=∏i∈E1Ui×∏i∉E1Ki and B2=∏i∈E2Vi×∏i∉E2Ki in ℬ,

B1∩B2 =∏i∈E1∖E2(Ui∩Ki)⁢∏i∈E2∖E1(Ki∩Vi)⁢∏i∈E1∩E2(Ui∩Vi)⁢∏i∉E1∪E2Ki.

Since Ki is open, Ui∩Ki and Ki∩Vi are open, and because E1∪E2 is finite we get that B1∩B2 belongs to ℬ. ∎

We prove that the restricted product is locally compact.111Anton Deitmar and Siegfried Echterhoff, Principles of Harmonic Analysis, second ed., p. 258, Lemma 13.3.1. This is a motivation for using this object.

Lemma 2.

X is locally compact.

Proof.

For x∈X, let E⊂I be finite with x∈Ki for i∉E. For i∈E, because Xi is locally compact there is a compact neighborhood Ni of xi, with Ui open, x∈Ui⊂Ni. Then the product ∏i∈ENi×∏i∉EKi is compact, ∏i∈EUi×∏i∉EKi is open, and

x∈∏i∈EUi×∏i∉EKi⊂∏i∈ENi×∏i∉EKi,

showing that X is locally compact. ∎

The following is part of the machinery of restricted products.222Anton Deitmar and Siegfried Echterhoff, Principles of Harmonic Analysis, second ed., p. 258, Lemma 13.3.1.

Lemma 3.

For nonempty disjoint sets A,B⊂I with I=A∪B, the topological spaces

∏^i∈IKi⁢Xi

and

(∏^i∈AKi⁢Xi)×(∏^i∈BKi⁢Xi).

are homeomorphic.

2 Adeles

For a nonempty set of primes S, define

𝔸S=∏^p∈Sℤp⁢ℚp,𝔸S=∏^p∉Sℤp⁢ℚp.

Because ℤp is a ring, 𝔸S is a ring. We prove that 𝔸S is a locally compact topological ring.333Anton Deitmar and Siegfried Echterhoff, Principles of Harmonic Analysis, second ed., p. 258, Theorem 13.3.2.

Lemma 4.

For any nonempty set S of primes, AS is a locally compact topological ring.

Proof.

For a,b∈𝔸S, let a+b∈U=∏p∈EUp×∏p∈S∖Eℤp, for Up open in ℚp. But (x,y)↦x+y is continuous ℚp×ℚp→ℚ, so each for p∈E there is an open neighborhood Vp of ap in ℚp and an open neighborhood Wp of bp in ℚp such that x+y∈Up for (x,y)∈Vp×Wp. Let

V=∏p∈EVp×∏p∈S∖Eℤp,W=∏p∈EWp×∏p∈S∖Eℤp.

which belong to ℬ. (a,b)∈V×W, and if (x,y)∈V×W then x+y∈U, which shows that (x,y)↦x+y is continuous 𝔸S×𝔸S→𝔸S.

Likewise, let a⋅b∈U=∏p∈EUp×∏p∈S∖Eℤp. Because (x,y)↦x⋅y is continuous ℚp×ℚp→ℚp, for each p∈E there are open ap∈Vp⊂ℚp and bp∈Wp⊂ℚp such that x⋅y∈Up for (x,y)∈Vp×Wp.

This shows that 𝔸S is a topological ring. Finally, by Lemma 2, 𝔸S is locally compact. ∎

Let 𝔸fin=∏^p<∞ℤp⁢ℚp, which is a locally compact topological ring. Finally let

𝔸=ℝ×𝔸fin=ℝ×∏^p<∞ℤp⁢ℚp,

which is also a locally compact topological ring, whose elements are called adeles.

3 Embedding the rationals in the adeles

Write Np={0,…,p-1}. ℚp⊂∏ℤNp. For x∈ℚp,

vp⁢(x)=inf⁡{k∈ℤ:x⁢(k)≠0}.

vp⁢(x)=∞ if and only if x=0.

|x|p=p-vp⁢(x).
ℤp={x∈ℚp:vp⁢(x)≥0}={x∈ℚp:|x|p≤1}.

For r∈ℚ write |r|∞=|r|. It is straightforward that for r∈ℚ, r≠0,

|r|∞⋅∏p<∞|r|p=∏p≤∞|r|p=1.

Let Er={p<∞:vp⁢(r)<0}={p<∞:|r|p>1}, which is finite. Thus it makes sense to define ι:ℚ→𝔸 by ι⁢(r)p=r for p≤∞. It is immediate that ι is one-to-one. Assign ℚ the discrete topology, and then ι is continuous. We shall prove that ι:ℚ→ι⁢(ℚ) is a homeomorphism.

Theorem 5 (Chinese remainder theorem).

Let S be a nonempty finite set of primes. For each p∈S suppose ep is a positive integer and cp is an integer. Then there is a unique x+∏p∈Spep⁢Z such that x+pep⁢Z=cp+pep⁢Z for all p∈S.

Proof.

Let x,y∈ℤ and suppose that x+pep⁢ℤ=cp+pep⁢ℤ and x+pep⁢ℤ=cp+pep⁢ℤ for p∈S. This means that for each p∈S, pep divides x-y, and for p,q∈S, p≠q, gcd⁡(pep,qeq)=1 so ∏p∈Spep divides x-y, meaning x+∏p∈Spep⁢ℤ=y+∏p∈Spep.

Now let N=∏p∈Spep and for p∈S let Np=p-ep⁢N. Then gcd⁡(Np,pep)=1 so there is some 1≤up≤pep-1 such that

Np⁢up≡1(modpep).

Let x=∑p∈Scp⁢Np⁢up∈ℤ. For p,q∈S, q≠p, Nq≡0(modpep), so x≡cp(modpep). In other words, x+pep⁢ℤ=cp+pep⁢ℤ. ∎

Theorem 6 (Weak approximation theorem).

Let S be a nonempty finite set of primes and for p∈S let xp∈Qp. For ϵ>0 there is some r∈Q such that

|r-xp|p<ϵ,p∈S.
Proof.

Let N=∏p∈Sp. For each p∈S let kp>1 such that p-kp⁢N<1. Then define yp=p-kp⁢N∈ℚp. |yp|p=|p-kp⁢p|=pkp-1, so

|ypn1+ypn-1|p=1|1+ypn|p≤1|yp|pn-1=1pn⁢(kp-1)-1→0,

and for q∈S with q≠p, |yp|q=|q|q=q-1, so

|ypn1+ypn|q=q-n|1+ypn|q≤q-n1-q-n→0.

For p∈S take rp∈ℚ with |rp-xp|p<ϵ. For n≥1 define

zn=∑p∈Srp⁢ypn1+ypn∈ℚ.

For p∈S, ∑q∈Srq⁢yqn1+yqn→rp in ℚp. Take np with |zn-rp|p<ϵ, and for n=max⁡{np:p∈S} and r=zn, for any p∈S we have

|r-xp|p=|r-rp+rp-xp|p≤max⁡(|r-rp|p,|rp-xp|p)<ϵ.

∎

Lemma 7.

Let S be a nonempty finite set of primes and for each p∈S suppose xp∈Zp. For any ϵ>0 there is some x∈Z such that

|x-xp|p<ϵ,p∈S.
Proof.

For each p∈S let yp∈ℤ≥0 with |yp-xp|p<ϵ. Take p-np<ϵ and let n=max⁡{np:p∈S}. By the Chinese remainder theorem, there is some x∈ℤ such that x+pn⁢ℤ=yp+pn⁢ℤ for each p∈S. Because pn divides x-yp, |x-yp|p≤p-n. Then for any p∈S,

|x-xp|p=|x-yp+yp-xp|p≤max⁡(|x-yp|p,|yp-xp|p)<ϵ.

∎
