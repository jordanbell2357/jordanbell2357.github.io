1 Introduction

In this note I am working out some material following Steve Shkoller’s MAT218: Lecture Notes on Partial Differential Equations. However, I have written out a number of details that were not in the original notes, and may thus have introduced errors that were not in the notes on which this is based.

Write 𝕋=ℝ/2⁢π⁢ℤ, and for 1≤p<∞,

∥f∥Lp=(12⁢π⁢∫𝕋|f⁢(t)|p⁢𝑑t)1/p.

Define

∥f∥Hk=(∑0≤j≤k∥∂xj⁡f∥L22)1/2.

If u is a distribution on 𝕋, ∂x⁡u is also a distribution on 𝕋, and in particular, if u∈L2⁢(𝕋) then ∂x⁡u is a distribution on 𝕋. But if u∈H2⁢(𝕋), for example, then ∂x2⁡u is an element of L2⁢(𝕋), rather than merely being a distribution.

Fix T>0. Let f∈L2⁢(0,T;L2⁢(𝕋)) and g∈H1⁢(𝕋); as H1⁢(𝕋)⊂C0⁢(𝕋), we can speak about the value of g at every point rather than merely almost all points.

For almost all t and for all x, define fn by

fn⁢(x,t)=∑k=-nnf^⁢(k,t)⁢ei⁢k⁢x,

and for all x define gn by

gn⁢(x)=∑k=-nng^⁢(k)⁢ei⁢k⁢x.

In other words, if Dn⁢(x)=∑k=-nnei⁢k⁢x, then

fn⁢(x,t)=(Dn*f⁢(⋅,t))⁢(x),gn⁢(x)=(Dn*g)⁢(x),

where

(ϕ*ψ)⁢(x)=12⁢π⁢∫𝕋ϕ⁢(y)⁢ψ⁢(x-y)⁢𝑑y.

2 Truncation

For each n, assume that there is some un∈C∞⁢(0,T;C∞⁢(𝕋)) such that for almost all t and for all x∈𝕋,

un⁢t⁢(x,t)-un⁢x⁢x⁢(x,t)=fn⁢(x,t), (1)

and for all x∈𝕋,

un⁢(x,0)=gn⁢(x).

We will thus obtain a formula for un. In fact we will not necessarily have un∈C∞⁢(0,T;C∞⁢(𝕋)), but once we have an expression for un we can determine the function space of which it is an element. We will then show that there is some u in a certain function space such that un⁢(x,t)=(Dn*u⁢(⋅,t))⁢(x) for all x and t.

For all t and x,

un⁢(x,t)=∑k∈ℤun^⁢(k,t)⁢ei⁢k⁢x.

Then (1) becomes the statement that for almost all t and for all x,

∑k∈ℤun^′⁢(k,t)⁢ei⁢k⁢x+∑k∈ℤk2⁢un^⁢(k,t)⁢ei⁢k⁢x=∑k=-nnf^⁢(k,t)⁢ei⁢k⁢x.

If |k|>n, then un^′⁢(k,t)+k2⁢un^⁢(k,t)=0, which is a linear ordinary differential equation, whose solution satisfies un^⁢(k,t)=e-k2⁢t⁢un^⁢(k,0). Since un⁢(x,0)=gn⁢(x), un^⁢(k,0)=0. Hence if |k|>n then un^⁢(k,t)=0. If |k|≤n, then for almost all t, un^′⁢(k,t)+k2⁢un^⁢(k,t)=f^⁢(k,t). The solution of this is, for all t and for all x,

un^⁢(k,t)=e-k2⁢t⁢gn^⁢(k)+e-k2⁢t⁢∫0tek2⁢s⁢fn^⁢(k,s)⁢𝑑s.

Hence, for all t and for all x,

un⁢(x,t)=∑k=-nn(e-k2⁢t⁢gn^⁢(k)+e-k2⁢t⁢∫0tek2⁢s⁢fn^⁢(k,s)⁢𝑑s)⁢ei⁢k⁢x.

We merely know that fn^⁢(k,t) is defined for almost all t, thus we only know for almost all t0∈(0,T) and for all x that un⁢t⁢(x,t0) exists. We do have that

un∈C0⁢(0,T;C∞⁢(𝕋)).

3 𝐻¹

For almost all t, multiply (1) by un⁢(x,t) and integrate over 𝕋. This is,

∫𝕋un⁢t⁢(x,t)⁢un⁢(x,t)⁢𝑑x-∫𝕋un⁢x⁢x⁢(x,t)⁢un⁢(x,t)⁢𝑑x=∫𝕋fn⁢(x,t)⁢un⁢(x,t)⁢𝑑x.

Integrating by parts this becomes

∫𝕋un⁢t⁢(x,t)⁢un⁢(x,t)⁢𝑑x+∫𝕋un⁢x⁢(x,t)⁢un⁢x⁢(x,t)⁢𝑑x=∫𝕋fn⁢(x,t)⁢un⁢(x,t)⁢𝑑x,

which is

π⋅∂t⁡12⁢π⁢∫𝕋un⁢(x,t)2⁢𝑑x+2⁢π⋅12⁢π⁢∫𝕋un⁢x⁢(x,t)2⁢𝑑x=∫𝕋fn⁢(x,t)⁢un⁢(x,t)⁢𝑑x.

Writing this using norms,

π⋅∂t⁡∥un⁢(⋅,t)∥L22+2⁢π⋅∥un⁢x⁢(⋅,t)∥L22=∫𝕋fn⁢(x,t)⁢un⁢(x,t)⁢𝑑x.

Integrating from 0 to t, for any t,

π⋅∥un⁢(⋅,t)∥L22-π⋅∥un⁢(⋅,0)∥L22+2⁢π⁢∫0t∥un⁢x⁢(⋅,s)∥L22⁢𝑑s=∫0t∫𝕋fn⁢(x,s)⁢un⁢(x,s)⁢𝑑x⁢𝑑s.

For almost all s,

∫𝕋|fn⁢(x,s)⁢un⁢(x,s)|⁢𝑑x = 2⁢π⋅12⁢π⁢∫𝕋|fn⁢(x,s)⁢un⁢(x,s)|⁢𝑑x
≤ 2⁢π⋅∥fn⁢(⋅,s)∥L2⁢∥un⁢(⋅,s)∥L2
≤ 2⁢π⁢(∥fn⁢(⋅,s)∥L222+∥un⁢(⋅,s)∥L222)
= π⋅∥fn⁢(⋅,s)∥L22+π⋅∥un⁢(⋅,s)∥L22.

It follows that for all t (not just almost all t)

π⋅∥un⁢(⋅,t)∥L22-π⋅∥un⁢(⋅,0)∥L22+2⁢π⁢∫0t∥un⁢x⁢(⋅,s)∥L22⁢𝑑s≤∫0tπ⋅∥fn⁢(⋅,s)∥L22+π⋅∥un⁢(⋅,s)∥L22⁢d⁢s,,

so, as un⁢(x,0)=gn⁢(x),

∥un⁢(⋅,t)∥L22+2⁢∫0t∥un⁢x⁢(⋅,s)∥L22⁢𝑑s≤∥gn∥L22+∫0t∥fn⁢(⋅,s)∥L22+∥un⁢(⋅,s)∥L22⁢d⁢s.

Let

y⁢(t)=∥un⁢(⋅,t)∥L22+2⁢∫0t∥un⁢x⁢(⋅,s)∥L22⁢𝑑s.

By the inequality we just established we have, for all t,

y⁢(t) ≤ ∥gn∥L22+∫0t∥fn⁢(⋅,s)∥L22⁢𝑑s+∫0t∥un⁢(⋅,s)∥L22⁢𝑑s
≤ ∥gn∥L22+∫0t∥fn⁢(⋅,s)∥L22⁢𝑑s+∫0ty⁢(s)⁢𝑑s.

By Gronwall’s inequality, we get

y⁢(t)≤(∥gn∥L22+∫0t∥fn⁢(⋅,s)∥L22⁢𝑑s)⁢et.

As ∥gn∥L2≤∥g∥L2 and ∥fn⁢(⋅,s)∥L2≤∥f⁢(⋅,s)∥L2 (these two facts follow from Parseval’s identity), it follows that

y⁢(t)≤(∥g∥L22+∫0t∥f⁢(⋅,s)∥L22⁢𝑑s)⁢et.

Therefore, if 0≤t≤T then

∥un⁢(⋅,t)∥L22+2⁢∫0t∥un⁢x⁢(⋅,s)∥L22⁢𝑑s ≤ (∥g∥L22+∫0t∥f⁢(⋅,s)∥L22⁢𝑑s)⁢eT
≤ (∥g∥L22+∥f∥L2(0,T;L2(𝕋)2)⁢eT
= M.

By Parseval’s identity,

∑k∈ℤ|un^⁢(k,t)|2+2⁢∫0t∑k∈ℤ|un⁢x^⁢(k,s)|2⁢d⁢s≤M,

hence for all t,

∑k∈ℤ|un^⁢(k,t)|2+2⁢∫0t∑k∈ℤk2⁢|un^⁢(k,s)|2⁢d⁢s≤M.

If k≤n≤m, then un^⁢(k,t)=um^⁢(k,t) for all t. Define u^⁢(k,t) by

u^⁢(k,t)=limn→∞⁡un^⁢(k,t)=uk^⁢(k,t).

Thus, for all t,

∑k∈ℤ|u^⁢(k,t)|2+2⁢∫0t∑k∈ℤk2⁢|u^⁢(k,s)|2⁢d⁢s≤M. (2)

Then, for some M′=M′⁢(f,g,T),

∫0T∑k∈ℤ|u^⁢(k,t)|2+∑k∈ℤk2⁢|u^⁢(k,t)|2⁢d⁢t≤M′.

It follows that for almost all t, there is some u∈H1⁢(𝕋) whose Fourier coefficients are u^⁢(k,t), and that we have

∫0T∥u⁢(⋅,t)∥H12⁢𝑑t≤M′.

We have

limn→∞⁡∫0T∥un⁢(⋅,t)-u⁢(⋅,t)∥H12⁢𝑑t=0,

i.e.

limn→∞⁡∥un-u∥L2⁢(0,T;H1⁢(𝕋))2=0.

4 𝐻²

Multiply (1) by un⁢x⁢x⁢(x,t) and integrate over 𝕋. We get, for almost all t,

∫𝕋un⁢t⁢(x,t)⁢un⁢x⁢x⁢(x,t)⁢𝑑x-∫𝕋un⁢x⁢x⁢(x,t)⁢un⁢x⁢x⁢(x,t)⁢𝑑x=∫𝕋fn⁢(x,t)⁢un⁢x⁢x⁢(x,t)⁢𝑑x.

As

∫𝕋un⁢t⁢(x,t)⁢un⁢x⁢x⁢(x,t)⁢𝑑x=-∫𝕋un⁢t⁢x⁢(x,t)⁢un⁢x⁢(x,t)⁢𝑑x=-12⁢dd⁢t⁢∫𝕋un⁢x⁢(x,t)2⁢𝑑x,

we have

-π⁢dd⁢t⁢∥un⁢x⁢(⋅,t)∥L22-2⁢π⁢∥un⁢x⁢x⁢(⋅,t)∥L22=∫𝕋fn⁢(x,t)⁢un⁢x⁢x⁢(x,t)⁢𝑑x.

-π⁢∥un⁢x⁢(⋅,t)∥L22+π⁢∥un⁢x⁢(⋅,0)∥L22-2⁢π⁢∫0t∥un⁢x⁢x⁢(⋅,s)∥L22⁢𝑑s=∫0t∫𝕋fn⁢(x,s)⁢un⁢x⁢x⁢(x,s)⁢𝑑x.

5 Solution

For all t we have u⁢(⋅,t)∈H1⁢(𝕋), and H1⁢(𝕋)⊂C0⁢(𝕋), so for all t and all x, u⁢(x,t) is defined. The Sobolev embedding tells us that if k>α+12 then Hk⁢(𝕋)⊂Cα⁢(𝕋). So, being specific, we have H1⁢(𝕋)⊂C1/4⁢(𝕋). It is a fact that if h∈Cα⁢(𝕋), α>0, then the partial sums of the Fourier series of h converge to h in the supremum norm.

For all t and for each k,

u^⁢(k,t)=e-k2⁢t⁢g^⁢(k)+e-k2⁢t⁢∫0tek2⁢s⁢f^⁢(k,s)⁢𝑑s.

It follows that, for all x,

u⁢(x,0)=limN→∞⁡∑|k|≤Ng^⁢(k)⁢ei⁢k⁢x.

On the other hand,

g⁢(x)=limN→∞⁡∑|k|≤Ng^⁢(k)⁢ei⁢k⁢x.

Thus for all x, u⁢(x,0)=g⁢(x).

We have

∥ut-ux⁢x-f∥L2⁢(0,T;L2⁢(𝕋)) ≤ ∥ut-un⁢t∥L2⁢(0,T;L2⁢(𝕋))
+∥ux⁢x-un⁢x⁢x∥L2⁢(0,T;L2⁢(𝕋))
+∥f-fn∥L2⁢(0,T;L2⁢(𝕋))
+∥un⁢t-un⁢x⁢x-fn∥L2⁢(0,T;L2⁢(𝕋)).

Each of the four norms has limit 0 as n→∞. Let me work out the first one. For almost all t,

ut^⁢(k,t)-un⁢t^⁢(k,t) = ∑|k|>n-k2⁢e-k2⁢t⁢g^⁢(k)-k2⁢e-k2⁢t⁢∫0tek2⁢s⁢f^⁢(k,s)⁢𝑑s
+e-k2⁢t⁢ek2⁢t⁢f^⁢(k,t)
= ∑|k|>n-k2⁢u^⁢(k,t)+f^⁢(k,t).

Then using Parseval’s identity,

∥ut-un⁢t∥L2⁢(0,T;L2⁢(𝕋))2 = ∫0T∑|k|>n|-k2⁢u^⁢(k,t)+f^⁢(k,t)|2⁢d⁢t
≤ 2⁢∫0T∑|k|>nk2⁢|u^⁢(k,t)|2+|f^⁢(k,t)|2⁢d⁢t.

Then,

∥ut-ux⁢x-f∥L2⁢(0,T;L2⁢(𝕋))2=0.

So, for almost all t and for almost all x,

ut⁢(x,t)-ux⁢x⁢(x,t)=f⁢(x,t).
