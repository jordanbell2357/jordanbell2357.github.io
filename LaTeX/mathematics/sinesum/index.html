George Pólya and Gábor Szegő, Problems and theorems in analysis, Volume II, volume 216 of Die Grundlehren der mathematischen Wissenschaften, Springer, 1976, translated from the German by C. E. Billigheimer.

The following is an estimate of the Lp norms of a sum with nonnegative terms (p. 77, no. 38).

Let Γn⁢(t)=∑k=1n|sin⁡k⁢t|k. As |sin⁡k⁢t|≤1, we have Γn⁢(t)≤∑k=1n1k, but we can give a sharper upper bound for Γn⁢(t) using the following two results. First, if Bn⁢(t)=∑k=1ncos⁡k⁢tk, then Bn⁢(t)≥-1 for all t (p. 75, no. 28). Second , for all t (p. 76, no. 34),

|sin⁡t|=2π-4π⁢∑j=1∞cos⁡2⁢j⁢t4⁢j2-1.

Therefore

Γn⁢(t) = ∑k=1n1k⁢(2π-4π⁢∑j=1∞cos⁡2⁢j⁢k⁢t4⁢j2-1)
= 2π⁢∑k=1n1k-4π⁢∑j=1∞14⁢j2-1⁢∑k=1ncos⁡2⁢j⁢k⁢tk
= 2π⁢∑k=1n1k-4π⁢∑j=1∞Bn⁢(2⁢j⁢t)4⁢j2-1
≤ 2π⁢∑k=1n1k+4π⁢∑j=1∞14⁢j2-1
= 2π⁢∑k=1n1k+2π,

using

∑j=1∞14⁢j2-1=∑j=1∞12⋅12⁢j-1-12⋅12⁢j+1=12.

Thus ∥Γn∥p≤∥Γn∥∞≤2π⁢∑k=1n1k+2π. On the other hand,

∥Γn∥1 = 12⁢π⁢∫02⁢πΓn⁢(t)⁢𝑑t
= 12⁢π⁢∑k=1n1k⁢∫02⁢π|sin⁡k⁢t|⁢𝑑t
= 12⁢π⁢∑k=1n1k2⁢∫02⁢π⁢k|sin⁡t|⁢𝑑t
= 12⁢π⁢∑k=1n1k⁢∫02⁢π|sin⁡t|⁢𝑑t
= 12⁢π⁢∑k=1n1k⋅4
= 2π⁢∑k=1n1k.

Therefore ∥Γn∥p≥∥Γn∥1=2π⁢∑k=1n1k.

Hence for 1≤p≤∞,

2π⁢∑k=1n1k≤∥Γn∥p≤2π+2π⁢∑k=1n1k.

So,

∥Γn∥p=2π⁢log⁡n+O⁢(1).
