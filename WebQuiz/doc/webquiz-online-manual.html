WebQuiz: Web Quizzes using LaTeX
Question Introduction ◄ Questions ►
WebQuiz is a LaTeX package for creating interactive web quizzes. The idea is that you write the quiz using LaTeX and that WebQuiz creates the web page from this file. Anything that you can write using LaTeX will be converted to <a href="https://www.w3schools.com/html/html_intro.asp" ><span style="color:Brown">HTML</span></a> by WebQuiz. This includes, for example, mathematics and graphics written using <a href="https://www.ctan.org/pkg/pstricks" ><span style="color:Brown">pstricks</span></a>. WebQuiz supports three different types of quiz questions: <dl class="enumerate-enumitem"><dt class="enumerate-enumitem"> 1. </dt><dd class="enumerate-enumitem">Multiple choice questions with a unique correct answer. (See <span class="button blank" onClick="gotoQuestion(1);">1</span>) </dd><dt class="enumerate-enumitem"> 2. </dt><dd class="enumerate-enumitem">Multiple choice questions with several (or no) correct answers. (See <span class="button blank" onClick="gotoQuestion(2);">2</span>) </dd><dt class="enumerate-enumitem"> 3. </dt><dd class="enumerate-enumitem">Questions that require the student to enter an answer, which can then be compared with the correct answer in several different ways. (See <span class="button blank" onClick="gotoQuestion(3);">3</span>)</dd></dl> <p>The use of WebQuiz is described in the next section. Later sections describe how each of the WebQuiz environments are used. </p> <p>The <span class="lstinline"></span><span style="color:ForestGreen">discussion</span> environment in WebQuiz can also be used to write Web Pages like this one (The pages you are reading here were written using WebQuiz.) </p>
Once you have a WebQuiz file, you can run it through LaTeX, in the usual way, to produce a readable version of your quiz. When you are happy with the quiz, use WebQuiz to create the HTML version. Note that the printable version of the quiz does not look like the web page; rather, it contains all of the information in an easily readable layout.
